# Hospital-Emergency-Room-Dashboard

Overview

This project is an interactive Excel dashboard created to analyze hospital emergency room data. It helps track patient flow, wait time, admission status, and overall performance.

Dataset

The dataset consists of patient-level hospital data stored in a single Excel sheet.

Key Fields:

* Patient ID
* Gender
* Age Group
* Admission Status
* Wait Time
* Department Referral
* Satisfaction Score
* Date (Month & Year)



Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning



Process

* Cleaned and structured raw patient data
* Created pivot tables for analysis
* Built KPIs:

  * Total Patients
  * Average Wait Time
  * Satisfaction Score
* Designed an interactive dashboard using slicers



Dashboard Features

* Month-wise filtering
* Year selection
* Patient distribution by age group
* Gender-wise analysis
* Admission vs non-admission comparison
* Department referral analysis
* Patient satisfaction tracking



Key Insights

* Nearly equal split between admitted and non-admitted patients (~50%)
* Majority of patients fall in the 20–39 age group
* General Practice receives the highest number of referrals
* Wait time fluctuations indicate peak load periods

Conclusion

This dashboard provides a clear and interactive way to monitor emergency room performance and supports better decision-making using data.
